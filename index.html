Unlocker GUI - File & Task Manager

Unlocker GUI is a simple desktop tool for managing files and tasks with administrator rights. It supports operations like deleting, renaming, moving files, or killing processes with a modern dual-language interface.

Features

  • Runs with Administrator privileges
  • Supports Arabic and English UI
  • Operations: Kill Process, Delete, Rename, Move File
  • Settings and logs saved in JSON
  • Polished user interface with icons
